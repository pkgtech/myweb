Laundry Kart

Laundry Service online platform.

Project Date May, 2020
Role Founder, Backend Developer, Core Maintainer
Tech Stack FrontEnd: Bootstrap, jQuery
BackEnd: PhP, MySQL, API's, javascript
Visit Beta

It's an website which takes orders online for Laundry and later processes it. Its is made using HTML/CSS/JQuery and PHP. This project sharpened my Project Management skills.

It is made using BOOTSTRAP 5

How its works :
1. User register/login
2. Redirected to dashboard
3. He can see his past orders and processing orders
4. He/She can book new appointment
5. And the circle moves on
