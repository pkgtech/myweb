File Parcel

It helps to transfer files between Android/iPhone and PC on same WIFI network.

Project Date Aug, 2020 ( Working on it )
Role Founder, Backend Developer, Core Maintainer
Tech Stack FrontEnd: Bootstrap, jQuery
BackEnd: Node.js, Docker, Firebase
Visit Not Released

There are some reasons for making this Idea:

Reason 1: While working at home which has WiFi network to which all devices are connected like mobile's,pc's. At time of sharing some files from mobile->pc or mobile->mobile <100/200 mb we have use Whatsapp or through XENDER(Banned so can't use). And in whatsapp we can't send >50mb files.

Reason 2: After getting banned of XENDER, which was too used while sharing files there should be an alternative. There are many but there is not one specific app available on everybody's phone like Xender.

Reason 3: Many time we have to transfer some files to PC so we connect USB cable or use whatsapp desktop. There is no convenient method to do.

Here comes the saviour.

File Parcel website helps to transfer files connected on same wifi network. The connection established would be P2P connection. Mainly useful at home/small organization which has wifi. And wifi is private network so external person interruption is not possible so there will be no security issues.

How to Use?

: Just visit our website
: All devices which has opened our website will be visible to each other.
: Tap on user and your file manager will open.
: choose a file.
: choose a file. Done!
